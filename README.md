# Manual QA Testing Portfolio

## Overview
This repository contains self-directed manual QA testing projects created to demonstrate foundational QA skills and readiness for an entry-level or intern QA role.

## Scope of Work
- Manual test case design and execution
- Functional, negative, and boundary testing
- Exploratory testing
- Bug reporting with severity and priority
- QA documentation using spreadsheets

## Applications Tested
- Sauce Demo (e-commerce demo application)
- The Internet by Herokuapp (QA practice site)

## Tools Used
- Excel / Google Sheets
- Google Chrome
- GitHub for version control
